## Imagen del Dispositivo Verba

Por favor, guarda la imagen del dispositivo Verba aquí con el nombre **`verba-device.png`**

La imagen debe ser:
- El dispositivo blanco circular con el ícono de micrófono en el centro
- Formato: PNG con transparencia (recomendado) o JPG
- Tamaño recomendado: 600x600px o superior
- Fondo: Transparente o blanco

Una vez guardada la imagen como `verba-device.png` en esta carpeta, aparecerá automáticamente en el hero section del landing page.
